Using Replay Debugging Mode : Create a Recording
Create a Recording
To debug an application during replay, you must have a recording of the application running. You could use the Workstation user interface to power on a virtual machine, copy your program from the host to the guest, share its application folder, start recording, run the program, wait for it to finish, stop recording, and update VMware > Options to designate this new recording. The Integrated Virtual Debugger automates these steps.
