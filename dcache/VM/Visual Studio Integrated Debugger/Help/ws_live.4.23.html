Using Live Debugging Mode : Debug Live Applications in Virtual Machines
Debug Live Applications in Virtual Machines
After you create the required configurations for live debugging, the Integrated Virtual Debugger lets you:
•
Start a live debugging session in a virtual machine.
•
Start an application in a virtual machine without debugging.
•
Start a debugging session that attaches to a process already running in a virtual machine.
