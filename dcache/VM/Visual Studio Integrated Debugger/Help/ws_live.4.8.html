Suppress Security Prompts on the Guest System
Running an application from a network share triggers a security prompt every time a shared file is accessed. VMware recommends that you turn off security prompts on the guest operating system.
You can use either Internet Explorer, or edit your registry key, to suppress security prompts.
To suppress security prompts on the guest system with Internet Explorer
1
In the Internet Explorer browser on the guest, select Tools > Internet Options > Security > Local Intranet and click Sites.
2
Click Advanced and add a new Web site:
file://*..host
To edit a registry key to suppress security prompts
1
In the guest system, open the registry with regedit.
2
Add a new key called .host under HKCU\Software\Microsoft\Windows\CurrentVersion\InternetSettings\ZoneMap\Domains.
3
In the .host key, create a DWORD Value called file and set its value to 1.
