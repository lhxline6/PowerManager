Set Post-Debug Event Properties
Post-Debug Event options determine which actions are performed after the debug command ends.
To set Post-Debug Event properties
1
Select VMware > Options.
2
In the left pane, click Live Debugging in VM and select Post-Debug Event.
3
You can set Command Line to one or more semicolon-delimited commands that are run after the debugging session ends.
4
You can the Termination Mode to do nothing, power off the guest, revert to the parent snapshot, or suspend the guest.
