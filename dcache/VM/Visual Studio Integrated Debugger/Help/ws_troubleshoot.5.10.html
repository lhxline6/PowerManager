Visual Studio 2005 Fails in Vista
Problem
Live debugging mode does not work in Visual Studio 2005 on a Windows Vista host.
Solutions
•
Manually configure the firewall to allow traffic from Visual Studio.
•
Run Visual Studio with Administrator permissions.
For more information about Visual Studio 2005 on Vista, go to the Microsoft Developer Network Web site and search for “Visual Studio 2005 on Windows Vista Issue List.”
