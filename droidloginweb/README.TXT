Pasos para instalacion:

1-Colocar esta carpeta droidlogin dentro del directorio htdocs dentro del servidor apache.
2-Modificar del archivo config.php  las siguiente lineas:

define("DB_USER", "root"); //cambiar por el nombre de usuario definido en la configuracion de la BD.
define("DB_PASSWORD", "sebabdroot");//Modificar por el password elegido

3-Al acceder a http://127.0.0.1/droidlogin/login.php deberiamos ver una ventana de login.
